# 串口输出日志

有了 shell 之后，直接将日志输出到控制台，开始有点乱了，可以将日志通过宏去掉，或者将日志输出到串口。

另外，bochs [^bochs][^serial] 好像对串口的支持不太好，目前不知道是为什么所以直接禁了。

## TTY

teletypewriter - 电传打字机
